# MusicZ - 极简专属桌面音乐盲盒

一款专为 Windows 系统深度打造的极简桌面端音乐播放器。抛弃繁琐的主动搜索与歌单管理，核心主打“音乐盲盒”式的随机探索体验，让每一次播放都充满未知的惊喜。

## ✨ 核心特性

* **纯粹极简，开箱即用**：零门槛体验，彻底免除繁琐的登录流程限制。
* **沉浸式盲盒体验**：打破听歌信息茧房，侧重于随机算法推荐，每次冷启动都能遇见不一样的声音。
* **内置强劲引擎**：底层原生融合网易云 API 引擎，无需用户手动寻找和配置任何 API Keys，直接畅听。
* **Windows 专属适配**：100vw/vh 贴合系统窗口，无滚动条、无边框干扰，带来极致纯粹的视听专注感。
